# Video Placeholders

Coloque os vídeos aqui:

## 1. studio-bg.mp4
- **Uso**: Background de baixa opacidade (30%)
- **Formato**: MP4 (H.264)
- **Duração**: ~30 segundos (loop)
- **Resolução**: 1080p ou 720p
- **Aspectos**: Horizontal (16:9)
- **Conteúdo**: THAMI no estúdio, ambiente low-light
- **Otimização**: Comprimir para web (~2-5MB)

## 2. cctv-feed.mp4
- **Uso**: Feed de câmera CCTV vertical
- **Formato**: MP4 (H.264)
- **Duração**: 15-30 segundos (loop)
- **Resolução**: 1080x1920 (vertical)
- **Aspectos**: Vertical (9:16)
- **Conteúdo**: Clip vertical do estúdio
- **Estilo**: Pode ter filtro preto e branco ou low saturation
- **Otimização**: Comprimir para web (~3-7MB)

Os vídeos ficarão em autoplay e loop.
